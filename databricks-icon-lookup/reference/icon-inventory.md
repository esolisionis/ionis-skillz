## URL Construction

**CDN Base**: `https://cdn.jsdelivr.net/gh/robkisk/databricks-icons@main/`

**For container icons** (root level):
```
https://cdn.jsdelivr.net/gh/robkisk/databricks-icons@main/general-icons/{filename}
```

**For folder-based icons** (URL encode spaces as `%20`):
```
https://cdn.jsdelivr.net/gh/robkisk/databricks-icons@main/general-icons/{Folder%20Name}/{filename}
```

**Examples**:
```
# Container icon (direct)
https://cdn.jsdelivr.net/gh/robkisk/databricks-icons@main/general-icons/unity-catalog-icon-full-color-container.svg

# Folder-based icon (encode spaces)
https://cdn.jsdelivr.net/gh/robkisk/databricks-icons@main/general-icons/Primary%20Icon%20Delta%20Lake/primary-icon-navy-delta-lake-1.svg
```

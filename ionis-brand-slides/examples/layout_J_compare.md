# Example — Layout J (Compare / contrast)

A worked example of the compare/contrast layout setting up a problem space with two cards (the "gap" between two existing things, with the third concept introduced on a later slide).

---

### Slide 2 — The Gap

**Layout:** Layout J (Compare / contrast — 2-card variant)

**Title:** "Prompts forget. Agents stay home."

**Body / copy regions:**

- **Lede above the cards:** "We've got great tools for one-shot tasks and great tools for in-platform agents. But there's a missing layer in between — the part that *travels with you*."

- **Card 1 — Prompt (neutral, Cool Gray 1 background):**
  - Badge: `PROMPT` (Cool Gray 11 fill, white text)
  - Headline: "Powerful, but one-shot."
  - Body: "A great prompt solves one problem in one chat. Close the window and it's gone. Share it as a screenshot and it rots."

- **Card 2 — Agent (qualified, Pink tint background):**
  - Badge: `AGENT` (Magenta fill, white text)
  - Headline: "Useful, but stuck." (in Raspberry)
  - Body: "Agents live inside one platform (Glean, ChatGPT, Claude…). When the tooling changes, the agent doesn't come with you."

**Visual spec:**
- Primary color anchor: Raspberry (in slide title's solid Raspberry rule, the Agent card's Magenta badge, the headline color)
- Prow block composition: none (Layout J uses rounded-corner cards, not Prows)
- Wing treatment: Cool Gray 1 watermark, left edge, alt-cropped, ~14% opacity
- Photography: none
- Typography accents: Raspberry on the Agent card's headline; the slide title carries the standard solid Raspberry rule

**Notes on this 2-card variant:** When using Layout J with only two cards (because the deck's third concept is unveiled on the following slide), Card 1 carries the neutral/gray treatment and Card 2 carries the pink-tinted/qualified treatment. The "answer" card (solid Raspberry, white text) is held in reserve for the slide that introduces the resolution.

**Speaker notes (optional):** "I'm setting up a gap — prompts on the left, agents on the right, and a third thing in the middle that we haven't talked about yet. That third thing is the rest of the deck."
